Rules:

  1. Try and move all of the images back to where they belong
  2. Start by pressing the shuffle button to move the pieces into random positions
  3. To move the pieces, select an image by clicking on it and select another image to swap them
  4. A selected image will have a red border
  5. Try to solve the puzzle in the least clicks possible; The higher the score, the worse it is
  6. Good luck!
